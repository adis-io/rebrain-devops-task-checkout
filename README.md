# Sample nginx config

This repo contains single `nginx` example configuration file

## Getting Started
### Prerequisites

You need pre-installed `nginx`. Installation depends on OS.

**Ubuntu**  
`sudo apt install nginx`

**OSX**  
`brew install nginx`

### Applying config

Copy nginx.conf to configurations directory

Latter depends on OS

**Ubuntu**  
`/etc/nginx/`

**OSX**  
`/usr/local/etc/nginx/`
